== Description == 

Adds UUIDs to WordPress posts.

== Installation ==

Place the SimpleUuid folder in the WP plugins directory. It should have the folder name "simple-uuid".

Go to the Plugins administrative page in WordPress and activate the plugin. Now whenever you save a post it will have a UUID placed in its meta data with the key 'simple-uuid'.
